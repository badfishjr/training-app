# No-Cost iPhone Install

The no-cost iPhone path is a home-screen web app. It behaves like an app, keeps your data on the phone, and does not require an Apple Developer account.

## Best Free Option

Host these files on a free static host, then open the HTTPS link on your iPhone and choose:

Safari -> Share -> Add to Home Screen

Good free hosts:

- GitHub Pages
- Cloudflare Pages
- Netlify

## Important

- A native App Store-style iPhone app usually requires Apple's paid developer account.
- The hosted web app can still work from your home screen and cache for offline use.
- Your workout logs are stored in the phone browser's local storage, not on a server.

## Files To Upload

Upload everything in this folder:

- `index.html`
- `app.js`
- `styles.css`
- `manifest.webmanifest`
- `sw.js`
- `icon.svg`
- `icon-180.png`
- `icon-192.png`
- `icon-512.png`
